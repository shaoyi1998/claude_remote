# Claude Remote - Claude Code 远程任务管理中心

## 项目简介
一个轻量的远程任务管理工具，用于管理和监控 Claude Code 会话。支持手机端访问，可查看任务进度、新建任务、与运行中的 Claude Code 进行完整交互。

## 核心功能
1. **任务列表** - 查看所有 Claude Code 会话及其运行状态
2. **任务监控** - 实时查看任务输出和进度
3. **新建任务** - 启动新的 Claude Code 会话
4. **完整交互** - 向运行中的任务发送指令继续对话
5. **任务控制** - 暂停、继续、终止任务

## 技术架构

│  手机 APK   │ ───▶ │  公网服务器       │ ───▶ │   家里电脑 WSL  │

                                                        ▼
                                               ┌─────────────────┐

                                               └─────────────────┘
                                                        │
                                                        ▼
                                               ┌─────────────────┐
                                               │  Claude Code    │

## 技术栈

### 后端
- **语言**: Python 3.10+
- **框架**: FastAPI (轻量、异步、自带 API 文档)
- **任务管理**: tmux + libtmux (Python 库)
- **认证**: JWT Token
- **数据库**: SQLite (轻量，单文件)

### 前端
- **框架**: Vue.js 3
- **UI 组件**: 适配手机端的响应式设计
- **打包**: WebView 套壳生成 APK

## 部署环境
- **运行位置**: WSL Ubuntu 22.04
- **进程管理**: tmux (后端服务 + Claude Code 会话)
- **网络访问**: 通过公网服务器 + WireGuard VPN 连接

## 安全设计
- 登录认证 (账号密码)
- JWT Token 验证
- 依赖 WireGuard 加密传输
